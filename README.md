这是一个基于 1 小时级别 EMA / MA 均线交叉的趋势跟随策略，逻辑清晰，适合放在 GitHub 作为基础量化示例。  核心思路：  使用 1H 的 EMA10 与 MA20 判断趋势方向  金叉做多，死叉做空  反向交叉即平仓，不持有双向仓位  严格一单一仓（pyramiding = 0），避免加仓放大回撤  策略特点：  多周期调用：可在任意周期运行，但信号来源固定为 1H  偏趋势型，适合震荡少、单边明显的行情  手续费已纳入回测，更接近真实交易环境  calc_on_every_tick = true，回测更精细，但结果更“残酷”  适用场景：  用作趋势策略模板  后续可扩展：止损 / 移动止盈 / 过滤震荡（如 ATR、ADX、结构过滤）
